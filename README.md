#   Anex / newEra Programming Language

Anex is a human-centric, symbol-free programming language designed for readability and simplicity. Stop fighting with `;` and `{}`—start writing code in English.

##   Installation
1. Go to the **Releases** tab on the right.
2. Download `AnexSetup.exe`.
3. Run the installer (Administrator rights required).
4. Open your terminal and type `anex --version`.

##   Syntax at a Glance
```anex
##
main:
    greeting is string.
    greeting is "Hello, World!".
    print 'greeting'.
,
##
